# Performance & Networking

EKS 환경에서의 네트워크 성능 최적화와 관련된 핵심 주제들을 다룹니다.

## 📋 주요 주제

### 🚀 DNS 성능 최적화
- CoreDNS 설정 최적화
- DNS 캐싱 전략
- 성능 모니터링 및 튜닝

### 🌐 네트워크 성능 향상
- Cilium ENI 모드 구현
- 네트워크 정책 최적화
- 트래픽 라우팅 개선

### 📊 모니터링 및 관찰성
- CoreDNS 메트릭 수집
- 네트워크 성능 측정
- 문제 진단 및 해결

이 섹션을 완료하면 다음을 할 수 있게 됩니다:

- EKS 클러스터의 DNS 성능을 최적화

- 네트워크 성능 모니터링 시스템 구축

## 개요

RAG 파이프라인과 장기 메모리가 정상 동작하려면 벡터 검색 인프라가 안정적으로 운영되어야 한다. 본 섹션은 Milvus 기반 벡터 데이터베이스 운영을 다룬다. 향후 Feature Store 운영, Knowledge Graph 인프라 등 데이터 계층 문서가 추가될 예정이다.

## 개요

AWS Labs AIDLC 방법론을 이해하고, AI 코딩 에이전트를 활용한 EKS 애플리케이션 개발 파이프라인을 구축합니다.

## 다룰 내용

- AIDLC 방법론 이해 (Inception → Construction → Operations)
- AI 코딩 에이전트 활용 (Amazon Q Developer, GitHub Copilot)
- 품질 게이트 및 AI 기반 코드 검증 시스템
- 기존 GitOps 워크플로우와의 통합 (ArgoCD + AI 리뷰)
- AIDLC 워크플로우 템플릿 및 자동화
- AI 에이전트 기반 인프라 코드 생성 (Terraform, Helm)

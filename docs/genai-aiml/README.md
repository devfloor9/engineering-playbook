# GenAI & AI/ML

EKS 환경에서 GenAI 및 AI/ML 워크로드를 효율적으로 운영하기 위한 전략과 구현 방법을 다룹니다.

## 📋 주요 주제

### 🤖 프로덕션 GenAI 구축
- LiteLLM, LangGraph, Langfuse 통합
- 확장 가능한 GenAI 아키텍처
- 모델 서빙 최적화

### ⚡ GPU 효율성 극대화
- MIG(Multi-Instance GPU) 활용
- Time-Slicing 전략
- 리소스 스케줄링 최적화

### 🏗️ GenAI 플랫폼 구축
- EBA 워크샵 가이드
- 엔드투엔드 파이프라인 구성
- 모니터링 및 관리 시스템

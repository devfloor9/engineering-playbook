## 데모 자산

### [LG H&H Marketing Innovation PoC](./lg-hnh-marketing-innovation/)
LG생활건강 **3 BU(Beauty + HDB + Refreshment) 통합** 마케팅 혁신 PoC — 자사 데이터(회원·SKU·캠페인) + 외부 4종 시그널(소셜·기상·경제·경쟁사) × **Ontology + Agentic AI**.

- **5 부서 페르소나**: 브랜드 마케터 / 소비자·트렌드 인사이트 / D&A·MarTech / CRM·LG 멤버스 / MD·채널 영업
- **25 KG 클래스**: 회원·상품(후/숨37/엘라스틴/페리오/코카콜라 등 3 BU)·거래·옴니채널·외부 4종
- **8 시나리오**: 시맨틱 검색·페르소나 챗봇·인사이트 카드·페르소나 매칭/클러스터링·캠페인 ROAS·외부 시그널 융합·옴니채널 여정·가드레일
- **AWS Stack**: Bedrock Sonnet 4.6 + Cohere embed/rerank + Neptune + OpenSearch Serverless + AgentCore + Glue + ECS Fargate ARM64

| 가치 | 설명 |

| **컨셉이 아닌 동작 시연** | 실데이터 N=500~5K + 합성 50K + 외부 라이브 |
| **부서 페르소나 스위처** | 동일 데이터, 부서별 시각 |

| **데이터 거버넌스** | Bedrock Guardrails + cohort_tag |
| **AWS Full Stack** | 8주 PoC 완성 |

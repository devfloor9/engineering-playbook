# Hybrid & Multi-Cloud

EKS를 클라우드 경계를 넘어 확장하고 하이브리드 및 멀티클라우드 환경에서 운영하는 방법을 다룹니다.

## 📋 주요 주제

### 🌐 하이브리드 노드 설정
- EKS 하이브리드 노드 구성
- 온프레미스 연결 설정
- 네트워크 구성 및 보안

### ☁️ 클라우드 버스팅
- 동적 워크로드 스케일링
- 하이브리드 워크로드 관리
- 비용 최적화 전략

### 🔗 멀티클라우드 연결
- 클라우드 간 네트워킹
- 데이터 동기화 및 복제
- 재해 복구 전략

## 🎯 학습 목표

이 섹션을 완료하면 다음을 할 수 있게 됩니다:

- EKS 클러스터를 온프레미스로 확장
- 하이브리드 워크로드 효율적 관리
- 멀티클라우드 아키텍처 설계 및 구현
- 클라우드 간 원활한 워크로드 이동

## 📚 문서 목록

클라우드의 경계를 넘나드는 현대적인 인프라 구축 방법을 배워보세요.

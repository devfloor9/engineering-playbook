# Observability & Monitoring

EKS 환경에서의 관찰 가능성 구현과 모니터링 시스템 구축에 대한 내용을 다룹니다.

## 📋 주요 주제

### 👁️ 네트워크 가시성
- Hubble을 통한 네트워크 관찰성
- 트래픽 플로우 분석
- 네트워크 정책 시각화

### 🤖 AI/ML 워크로드 모니터링
- Langfuse 통합 및 설정
- 모델 성능 추적
- 리소스 사용량 모니터링

### 📊 메트릭 및 로깅
- 프로메테우스 통합
- 그라파나 대시보드 구성
- 로그 수집 및 분석

## 🎯 학습 목표

이 섹션을 완료하면 다음을 할 수 있게 됩니다:

- EKS 클러스터의 네트워크 가시성 확보
- AI/ML 워크로드 모니터링 시스템 구축
- 종합적인 관찰 가능성 플랫폼 구현
- 성능 이슈 사전 감지 및 대응
